# Tablesmit — Product Hunt Description (60 words)

I needed one clean table for a report. I spent two hours fighting tools
that couldn't resize a column, refused to merge cells, and exported files
I was embarrassed to share.

So I built Tablesmit — a minimalist table maker with Excel-grade control
in the browser. Drag-to-resize. Merge any cells. Export to PDF, Excel,
CSV, or PNG. No account. Open source. Made in Nigeria.
